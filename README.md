# Kids

Kids is a Flutter educational app in Arabic for kindergarten children.

## Features

- Cognitive, linguistic, motor, and social-emotional activities.
- Multiple mini-games with progressive levels.
- Local stats tracking per game.
- Offline local voice playback fallback (`assets/voices/tts`) with TTS fallback.

## Run

```bash
flutter pub get
flutter run
```

## Build APK

```bash
flutter build apk --release
```
